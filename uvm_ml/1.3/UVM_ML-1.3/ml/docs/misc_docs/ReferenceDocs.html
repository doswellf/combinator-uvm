UVM-ML Reference Documentation

 

This reference documentation provides information on the files, types, and functions of the UVM-ML library.

Material is organized in the following topics:

 

·         Backplane provided API                (API implemented by the backplane for the frameworks)

·         Backplane required API                (API that must be implemented by the frameworks)

 

·         UVM-SC framework

·         UVM-SC adapter 

 

Note: this material is extracted automatically from the code files. The documentation will be further enhanced in future releases.
